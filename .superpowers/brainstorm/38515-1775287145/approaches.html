Tenant Detail Screen — Layout Approaches

3 ways to restructure after removals and additions. Changes: remove Manual Payment form, remove Manual Invoice form, remove Support Notes · Add: click-to-open details modal, Send Bill via SMS modal.

A

Minimal Restructure

Remove the 3 sections, wire up click handlers, add a "Send Bill" button next to the existing SMS section. Screen gets shorter but layout stays the same.

Section order (top → bottom)
Tenant Profile · Status Control
Tenant Settings
Recent Invoices (3, clickable) · Last Payments (3, clickable) · Receipts (3, clickable)
Customer Stats · Subscription
Engagement Snapshot · Recent Billing · SMS Top Up
Tenant Communication + Send Bill button added here
Support Notes — removed
Manual Invoice form — removed
Manual Payment form — removed

Pros

  • Lowest risk, fewest changes
  • Familiar order for existing users

Cons

  • Billing Queue + Recent Invoices are still duplicate sections
  • SMS Top Up and Send Bill are separated by distance
B

Consolidated Billing Row

Merge the duplicate "Recent Billing" and "Recent Invoices" cards into one. Group SMS Top Up + Send Bill together as a Communication card. Cleaner, less redundancy.

Section order (top → bottom)
Tenant Profile · Status Control
Tenant Settings
Last 3 Invoices (clickable) · Last 3 Payments (clickable) · Last 3 Receipts (clickable)
Customer Stats · Subscription · Engagement Snapshot
SMS Top Up card · Send Bill via SMS card (side by side)
Tenant Communication (general SMS)

Pros

  • Removes duplication (Billing Queue merged into Invoices card)
  • SMS actions grouped together
  • Shorter scroll

Cons

  • Moderate restructure — more lines to change
C

Tab-Based Layout

Split the screen into tabs: Overview, Billing, Communication. Each tab is focused and short. No more endless scroll.

Tab structure
Overview Billing Communication
Overview tab:
Profile · Stats · Subscription · Engagement · Counters
Billing tab:
Last 3 Invoices · Last 3 Payments · Last 3 Receipts (all clickable)
Communication tab:
SMS Top Up · Send Bill via SMS · General SMS

Pros

  • No scrolling — everything above the fold per tab
  • Cleanest structure

Cons

  • Biggest change — may feel unfamiliar
  • Context is split across tabs
